# power-supply-wiring
Разветвитель для лабораторного источника питания



Требования к выводам


- 2 стабилизированных USB выхода (5V, 1.5A) с независимым стабилизатором. Джамперное подключение bypass/regulator
- 1 нестабилизированный USB выход (bypass)
- 2 выхода на щупы
- 1 выход на "крокодилы"
